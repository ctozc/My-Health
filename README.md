# 健康管理系统 My Health(GUI)

- ## 概述
  基于javaFX及JFoenix的健康管理系统，PC端应用，包含GUI界面，技术栈：maven、javaFX、JFoenix、MySQL、Spring、MyBatis

    - #### Navicat
