# 健康管理系统 My Health(GUI)

- ## 概述
  本项目使用maven进行项目构建管理，使用javaFX和JFoenix实现GUI界面，使用mysql数据库，web框架使用mybatis、spring
------

- ## 环境
  - ### 开发环境
    - #### JDK:1.8
    - #### mysql

  - ### 开发工具
    - #### Intellij IDEA
    - #### SceneBuilder
    - #### Navicat

  - ### 开发框架
    - #### JavaFX
    - #### Spring
    - #### Mybatis
    - #### Maven
------
